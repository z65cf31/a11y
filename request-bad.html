Annotations

01. Content does not provides a logical and hierarchical heading structure

The page suggests styling for headings without use of the corresponding markup within the HTML code. The order and nesting of the headings used is not correct and may confuse people.

Other things to consider, in order to make headings more usable for screen reader users:

  • All pages should start by using an <h1> element.
  • Lists and headings used together to mark up content should be avoided
  • Be consistent in the use of headings

Note! Improvements on the template are propagated throughout the entire website
